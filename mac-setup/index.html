Homebrew

Super awesome package manager for OS X.


Prerequisites:

  • Xcode
  • Open Xcode at least once (to agree to Apple's license)
  • Download Command Line Tools

Installation

  • ruby -e "$(curl -fsSL https://raw.github.com/mxcl/homebrew/go)"
  • brew update
  • Now you're ready to start brewing!

http://mxcl.github.io/homebrew/

Brew Cask

No more dragging and dropping to install programs.


brew cask install dropbox

brew cask install virtualbox

brew cask install google-chrome

brew cask install sublime-text

brew cask linkapps --appdir="/Applications"


https://github.com/phinze/homebrew-cask
